# 🧭 ATEC 인사관리(근태) 자동화 프로그램

> **ATEC 인사관리팀**에서 사용하는 근태 자동화 툴로,  
> 그룹웨어와 ERP 데이터를 통합하여 근태현황·근태정리·익월적용 시트를 자동 생성하고,  
> 전월 대비 리포트 및 수당 계산을 시각화합니다.

---

## 🚀 주요 기능

| 구분 | 기능 설명 |
|------|------------|
| **1. 업로드** | RAW 데이터(시간외근무 승인현황)와 통상시급 파일 업로드. 전월 자료 선택 시 자동 비교 |
| **2. 리포트** | 전월 대비 근무시간·수당 증감률 시각화 (Chart.js 기반 막대/도넛 그래프) |
| **3. 세부내역** | 근태현황, 근태정리, 익월적용 3종 테이블 자동 생성 |
| **4. 데이터 저장** | Excel(.xlsx) 다운로드 — 개별 또는 통합 저장 기능 |
| **🧮 자동 계산** | VBA 기반 근태정리 로직 복원 (평일연장, 평일야간, 휴일일반, 휴일연장, 휴일연장야간, 수당 등) |
| **🧩 헤더 매핑 기능** | 업로드 엑셀의 컬럼명을 자동 인식하거나 수동 매핑 가능 |

---

## 🧠 계산 로직 요약

| 항목 | 기준 |
|------|------|
| **평일연장** | 평일 18:00 ~ 22:00 |
| **평일야간** | 평일 22:00 ~ 익일 06:00 |
| **휴일일반** | 휴일 06:00 ~ 22:00 |
| **휴일연장** | 휴일일반 중 8시간 초과 |
| **휴일야간** | 휴일 22:00 ~ 익일 06:00 |
| **휴일연장야간** | 휴일 중 22시 이후 8시간 초과분 |
| **적용시간** | 평일연장 + 평일야간 + 휴일일반 + 휴일연장 + 휴일야간 + 휴일연장야간 |
| **익월적용** | 종결일자 공란인 건은 당월 제외 및 익월 데이터로 분리 |

## 🧩 기술 스택

| 항목 | 내용 |
|------|------|
| **Frontend** | HTML5, CSS3, JavaScript (Vanilla) |
| **Library** | [Chart.js](https://www.chartjs.org/), [xlsx.js](https://github.com/SheetJS/sheetjs), [FileSaver.js](https://github.com/eligrey/FileSaver.js) |
| **데이터 파서** | `parseTime()` 함수로 시간문자열 → 실수 시간 변환 |
| **차트 플러그인** | `chartjs-plugin-datalabels` |
| **로컬 저장** | FileSaver로 `.xlsx` 생성 (SheetJS 활용) |
